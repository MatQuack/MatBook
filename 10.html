Mozilla Firefox (МФА: [moʊˈzɪlə ˈfaɪ(ɹ)fɑks]Перейти к разделу «#Название») — свободный браузер на движке Quantum, разработкой и распространением которого занимается Mozilla Corporation. Четвёртый по популярности браузер в мире (после Google Chrome, Safari и Microsoft Edge)[17] и первый среди свободного ПО — в марте 2021 года его рыночная доля составила ▼7,49 %[18]Перейти к разделу «#Популярность». В России Firefox занимает четвёртое место (после Google Chrome, Яндекс.Браузер и Opera) по популярности среди браузеров для ПК с долей 5,26 % пользователей на март 2021 года[19]. В браузере присутствует интерфейс со многими вкладками, проверка орфографии, поиск по мере набора, «живые закладки», менеджер загрузок, поле для обращения к поисковым системамПерейти к разделу «#Возможности». Новые функции можно добавлять при помощи расширений[20]Перейти к разделу «#Расширения». Firefox официально выпускается для Windows, macOS, GNU/Linux, Android и iOS. Доступны неофициальные сборки для FreeBSD, множества других UNIX-подобных операционных систем, а также BeOS. Начиная с версии 53.0, прекращена поддержка операционных систем Windows XP, Windows Vista, Windows Server 2003[23]. 5 сентября 2018 года была выпущена ESR-версия 60.2, что ознаменовало полный переход c Gecko на движок Quantum.
